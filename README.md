# 🛡️ SafeHer — Women Safety App

A Personal Emergency Response System for Women built with React Native + Expo.

## Features
- 🚨 One-tap SOS Alert with real-time location
- 📳 Shake to SOS — no screen touch needed
- ⏱️ Safety Timer with auto-alert
- 🔋 Low Battery Auto-Alert
- 📍 Live Location Tracking
- 👥 Trusted Contacts Management
- 🔐 Secure Login & Registration

## Tech Stack
React Native · Expo · TypeScript · expo-location · expo-sms · expo-sensors · expo-battery · react-native-maps · AsyncStorage

## Setup
```bash
npm install
npx expo start
```

## Platform
Android & iOS
